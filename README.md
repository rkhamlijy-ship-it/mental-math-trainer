# Mental Math Trainer  
A simple GUI game to practice mental math skills.

---

## Why this project  
Mental Math Trainer was created to demonstrate clean Python code, GUI development with Tkinter, and basic testing practices. It is intentionally simple, focusing on clarity, maintainability, and user interaction.

## Tech Stack  
- Python 3.x  
- Tkinter (GUI)  
- Pytest (testing)  
- Virtual environment (venv)

---

# English

## Description  
Mental Math Trainer is a small graphical game to improve your mental calculation skills. Solve math problems, get immediate feedback, and track your performance.

## Installation  
Clone the repository and navigate into the folder:  
```bash
git clone <your-repo-url>
cd mental-math-trainer
```

Create a virtual environment:  
```bash
python -m venv venv
```

Activate the virtual environment:  
Windows:  
```bash
venv\Scripts\activate
```  
macOS / Linux:  
```bash
source venv/bin/activate
```

Install dependencies (if any):  
```bash
pip install -r requirements.txt
```

## How to run  
```bash
python main.py
```

## Features  
- Choose difficulty: easy, medium, hard  
- Solve 5 math problems per round  
- Immediate feedback: correct, wrong, or invalid  
- Play again or exit  

## Testing  
Run tests with:  
```bash
pytest
```
